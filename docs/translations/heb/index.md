## בנייה

### בניית פיתוח

```bash
# שיבוט עם תת-מודולים
git clone --recursive https://github.com/PoC-Consortium/bitcoin-pocx.git
cd bitcoin-pocx/bitcoin

# תצורה עם PoCX מופעל
cmake -B build -DENABLE_POCX=ON

# בנייה
cmake --build build -j$(nproc)
```

**וריאנטי בנייה**:
```bash
# עם ממשק Qt גרפי
cmake -B build -DENABLE_POCX=ON -DBUILD_GUI=ON

# בניית Debug
cmake -B build -DENABLE_POCX=ON -DCMAKE_BUILD_TYPE=Debug
```

**תלויות**: תלויות בנייה סטנדרטיות של Bitcoin Core. ראו [תיעוד בניית Bitcoin Core](https://github.com/bitcoin/bitcoin/tree/master/doc#building) לדרישות ספציפיות לפלטפורמה.

### גרסאות שחרור

לקבצים בינאריים משוחזרים, השתמשו במערכת בניית GUIX: ראו [bitcoin/contrib/guix/README.md](../bitcoin/contrib/guix/README.md)
