## Σχετικά με αυτή την Τεκμηρίωση

Αυτή είναι η πλήρης τεχνική τεκμηρίωση για το Bitcoin-PoCX, μια ενσωμάτωση στο Bitcoin Core που προσθέτει υποστήριξη συναίνεσης Proof of Capacity neXt generation (PoCX). Η τεκμηρίωση είναι οργανωμένη ως πλοηγήσιμος οδηγός με διασυνδεδεμένα κεφάλαια που καλύπτουν όλες τις πτυχές του συστήματος.

**Κοινό-στόχος**:
- **Διαχειριστές Κόμβων**: Κεφάλαια 1, 5, 6, 8
- **Εξορύκτες**: Κεφάλαια 2, 3, 7
- **Προγραμματιστές**: Όλα τα κεφάλαια
- **Ερευνητές**: Κεφάλαια 3, 4, 5

Μεταφράσεις: [🇸🇦 Αραβικά](../ara/index.md) [🇫🇷 Γαλλικά](../fra/index.md) [🇨🇳 Κινέζικα](../zho/index.md) [🇩🇪 Γερμανικά](../deu/index.md) [🇯🇵 Ιαπωνικά](../jpn/index.md) [🇰🇷 Κορεατικά](../kor/index.md) [🇪🇸 Ισπανικά](../spa/index.md) [🇻🇳 Βιετναμέζικα](../vie/index.md)

## Συμβάσεις Τεκμηρίωσης

**Αναφορές Αρχείων**: Οι λεπτομέρειες υλοποίησης αναφέρουν πηγαία αρχεία ως `path/to/file.cpp:line`

**Ενσωμάτωση Κώδικα**: Όλες οι αλλαγές επισημαίνονται με `#ifdef ENABLE_POCX`

**Διασταυρούμενες Αναφορές**: Τα κεφάλαια συνδέονται με σχετικές ενότητες μέσω σχετικών συνδέσμων markdown

**Τεχνικό Επίπεδο**: Η τεκμηρίωση προϋποθέτει εξοικείωση με το Bitcoin Core και ανάπτυξη σε C++

## Μεταγλώττιση

### Έκδοση Ανάπτυξης

```bash
# Κλωνοποίηση με submodules
git clone --recursive https://github.com/PoC-Consortium/bitcoin-pocx.git
cd bitcoin-pocx/bitcoin

# Διαμόρφωση με ενεργοποιημένο το PoCX
cmake -B build -DENABLE_POCX=ON

# Μεταγλώττιση
cmake --build build -j$(nproc)
```

**Παραλλαγές Μεταγλώττισης**:
```bash
# Με Qt GUI
cmake -B build -DENABLE_POCX=ON -DBUILD_GUI=ON

# Έκδοση Debug
cmake -B build -DENABLE_POCX=ON -DCMAKE_BUILD_TYPE=Debug
```

**Εξαρτήσεις**: Τυπικές εξαρτήσεις μεταγλώττισης Bitcoin Core. Δείτε την [τεκμηρίωση μεταγλώττισης Bitcoin Core](https://github.com/bitcoin/bitcoin/tree/master/doc#building) για απαιτήσεις ανά πλατφόρμα.

### Εκδόσεις Παραγωγής

Για αναπαραγώγιμα εκτελέσιμα αρχεία, χρησιμοποιήστε το σύστημα μεταγλώττισης GUIX: Δείτε [bitcoin/contrib/guix/README.md](../bitcoin/contrib/guix/README.md)
